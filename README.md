# Probaho Browser

Probaho Browser is a lightweight custom web browser project.

## Features
- Fast browsing
- Minimal interface
- Custom build scripts

## Build Instructions

### Windows
Run:

build-browser.bat

### Linux / Mac

Run:

./build-browser.sh
